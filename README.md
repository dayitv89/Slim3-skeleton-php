# Slim3-skeleton-php

 `php composer.phar create-project slim/slim-skeleton [my-app-name]`

How to run 
 `http://localhost/Slim3-skeleton-php/`

 `cd [my-app-name]; php -S 0.0.0.0:8080 -t public index.php`
